# Restaurant Data Analysis Dashboard

Hi! I’m Shourya Parashar, and this is my interactive **Restaurant Data Analysis Dashboard** built using **Streamlit**.  
The goal of this project is to explore restaurant data, visualize trends, and demonstrate basic data analysis and feature engineering skills in a simple, recruiter-friendly dashboard.

---

## About the Project
This dashboard allows users (or recruiters) to interactively explore a restaurant dataset with **no additional setup required**.  
All analyses are preloaded with the dataset, so anyone can open the live app and immediately see the results.

The dashboard includes the following analyses:

- Data Exploration – Check missing values and data types.  
- Descriptive Analysis – View summary statistics for numeric columns.  
- Geospatial Analysis – Map restaurants by city using latitude and longitude.  
- Table Booking Analysis – See percentage of restaurants offering table booking.  
- Price Range Analysis – Bar chart of restaurant price ranges.  
- Feature Engineering – Create new features like restaurant name length.

---

## Dataset
- The dashboard uses `dataset.csv` (included in this repo).  
- Columns in the dataset include:  
  - `Restaurant Name`  
  - `City`  
  - `Latitude` & `Longitude`  
  - `Has Table booking`  
  - `Price range`  

The dataset is fully integrated into the dashboard, so **no need to upload anything**.

---

## How to Use
1. Open the live **Hugging Face Spaces link** for the dashboard.  
2. Explore each section by clicking the **expandable panels**.  
3. Interact with maps, charts, and feature analyses directly.  

Everything runs instantly, giving a smooth and recruiter-friendly experience.

---

## Tech Stack
- Python 3  
- Streamlit for the dashboard  
- Pandas for data handling  
- Matplotlib for charts  
- Scikit-learn for feature engineering
